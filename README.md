## About

**Random User Explorer** is a modern frontend web application that generates realistic user profiles using the **RandomUser API**.

The project demonstrates how to integrate external REST APIs using JavaScript's **Fetch API**, process JSON data asynchronously, and dynamically render user information with a responsive and visually appealing interface.

It serves as an excellent beginner-friendly project for understanding API consumption, asynchronous programming, and DOM manipulation.

---

## Features

-  Generate unlimited random user profiles
-  Display profile picture and personal details
-  Show email, phone number, and address
-  Display city, country, and postal code
-  Show age and date of birth
-  Fetch data using JavaScript Fetch API
-  Generate a new user with one click
-  Graceful error handling
-  Responsive design for desktop and mobile devices
-  Clean and modern user interface

## Getting Started

### Clone the repository

```bash
git clone https://github.com/kirtisharma-sudo/Global-People-Explorer.git
```

### Navigate into the project

```bash
cd Global-People-Explorer
```

### Run the project

Simply open **index.html** in your preferred web browser.
No additional installation is required.
